# Виртуальная машина и ассемблер для звездной анимации

## Описание проекта

Проект представляет собой полноценную виртуальную машину с собственным ассемблером, процессором и стековой архитектурой. Реализована анимация взмахов крыльев птицы и вывод текста "МАТАН" на псевдографическом экране 40×40.

## Архитектура

stars.asm (исходный код)
    ↓ ассемблер (asm.cpp)

bite_code.txt (байт-код)
    ↓ процессор (processor.cpp)
    
видеопамять (RAM[0..1599]) → вывод в консоль

## Компиляция и запуск

### Сборка

bash

chmod +x build.sh

./build.sh

Скрипт создаёт два исполнительных файла:

a.out — ассемблер (транслирует .asm в байт-код)

b.out — процессор (исполняет байт-код)

### Запуск

bash

### 1. Ассемблирование

./a.out stars.asm bite_code.txt

### 2. Исполнение

./b.out bite_code.txt

Название регистров аналогичны названия в x86-64 архитектуре. Все регистры являются регистрами общего назвначения

Программа stars.asm


## Технические детали

### Стек

Защита канарейками (LEFT_BIRD = 2147483647, RIGTH_BIRD = 2147483646)

Динамическое расширение/сжатие (x2 / ÷2)

stack_dump() для отладки

### Процессор

Регистры: 16 шт. (int regs[16])

RAM: 5000 ячеек

Два стека: основной stk и стек возвратов ret_stk

instruction_pointer — счётчик команд

### Ассемблер

Двухпроходная трансляция (для меток :label)

Поддержка меток и чисел как операндов переходов

Комментарии ;

Обработка ошибок

Программа проверяет:

Открытие/закрытие файлов

Выделение памяти

Корректность регистров

Деление на ноль

Переполнение стека

Коды ошибок определены в Stack_Error_Code и File_Usimng_Error_Code.

Проект разработан в учебных целях — демонстрация работы стековой виртуальной машины, динамической компиляции и ассемблера с полным циклом исполнения.
